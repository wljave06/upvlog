OPTION 1: With R2 Storage (Production - Recommended)
-----------------------------------------------------
1. Create R2 bucket:
   npx wrangler r2 bucket create vlog-videos

2. Push to GitHub/GitLab

3. Deploy via Cloudflare Dashboard:
   - Pages -> Create project
   - Connect Git repository
   - Build settings: leave empty
   - Deploy

4. Add R2 binding in Pages settings:
   - Functions tab
   - R2 bucket bindings
   - Variable: VIDEO_BUCKET
   - Bucket: vlog-videos

5. Done! Videos will be uploaded to R2 automatically


OPTION 2: Direct Upload (Static Files Only)
---------------------------------------------
1. Go to Cloudflare Pages -> Upload assets
2. Upload all files
3. Note: This won't support actual file uploads,
   only demo mode with localStorage

ğŸ¯ HOW IT WORKS
========================================

LOCAL DEVELOPMENT:
- Videos uploaded via /api/upload endpoint
- Files saved to videos/ folder
- Served with streaming support

PRODUCTION (Cloudflare):
- Videos uploaded to R2 bucket
- Served via CDN with caching
- Automatic streaming support

FALLBACK:
- If upload fails, uses localStorage
- Demo mode only (limited storage)
